搭建博客过程

【超详细！0 成本搭建个人网站！！【建议收藏】】 https://www.bilibili.com/video/BV1ts4y1f7Gu/?share_source=copy_web&vd_source=a4d260c12ae4261ed3a54b3afdb1e657

hexo 是一个非常简单、轻量级能够用来搭建个人博客的框架，在借助了小宇视频的基础上，本人打算基于这一框架进行博客的搭建。

分别安装 node.js, git 后，运行下列的命令，就可以很轻松的搭建出一个建立在 localhost 上的博客啦

将博客发布在 gitee pages 上

这一步的过程相对而言稍微麻烦一点

  1. 需要gitee进行实名认证
  2. 在本地创建公钥，并上传到ssh公钥
  3. ssh -T gitee@git.com
  4. hexo d(eploy)部署到gitee.com上
  5. 每次更新后hexo g(enerate) 然后hexo d
